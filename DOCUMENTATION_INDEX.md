## 🎓 Learning the Codebase

### Easy Entry Points (Start Here)

1. **App Structure**
   - Check: `app/` folder organization
   - File: Browse `app/dashboard/page.tsx`

2. **Database Schema**
   - File: `prisma/schema.prisma`
   - Watch for: Model relationships, enums

3. **Server Actions (How mutations work)**
   - File: `actions/leads.ts`
   - Watch for: Authorization check, Zod validation, activity logging

4. **Forms (How input works)**
   - File: `app/dashboard/leads/new/page.tsx`
   - Watch for: React Hook Form, Zod resolver, error handling

5. **Components (UI building blocks)**
   - Folder: `components/ui/`
   - Pattern: Same shadcn-ui style

### Intermediate (Go Deeper)

6. **Authentication**
   - File: `lib/auth.ts`
   - File: `middleware.ts`

7. **Validation**
   - File: `lib/validation.ts`
   - Pattern: Type-safe schemas

8. **Activity Logging**
   - File: `actions/activity.ts`
   - Pattern: Audit trail on all mutations
